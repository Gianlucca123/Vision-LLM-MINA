# VLM Model Downloader

This repository provides scripts to download and set up various VLM models. Each technique (online or local) has corresponding scripts that must be executed to download and prepare the models for use.

## Instructions

1. **Install Requirements**  
   Before proceeding, ensure that all required dependencies are installed. You can do this by running:
   ```bash
   pip install -r requirements.txt
   ```

2. **Download via local files**  
   Follow the instructions of the `localDownloadModels/README.md` file.

3. **Download via online files**  
   Follow the instructions of the `onlineDownloadModels/README.md` file.

## Notes
Make sure you have sufficient disk space for downloading the models.

Have fun!
