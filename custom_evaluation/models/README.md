# VLM Models

This repository contains all scripts related to various VLM models.

## Three repositories : 
### Call Models 
The `callModels/` directory contains functions to load and question various models used by `questionModel.py`.

### Download Models 
The `downloadModels/` directory contains scripts to download different models from external or local sources. 

See `downloadModels/README.md` for more information.

### Test Models 
The `testModels/` directory contains scripts to test each model. 

See `testModels/README.md` for more information.
